为什么是丹秘

跑在你自己的浏览器里

不是另起一个干净的浏览器，而是直接用你 Mac 上那个已经登录好的 Chrome。内网系统、需要登录态的页面，AI 打开就能读、能操作。

01

复用登录态，不交密码

AI 操作的是你 Mac 上那个已经登录好的 Chrome。Cookie、SSO、内网鉴权全都现成，你不用把任何凭据告诉 AI。

02

远程驱动，零本地依赖

丹秘在服务器侧调一个 HTTP 命令，服务器经加密 WebSocket 转发到你的扩展执行。你这边只装一个轻量扩展。

03

任务即分组，互不打架

每个任务是一个独立 session，对应 Chrome 里一个彩色标签分组。多个任务并行跑，视觉上清清楚楚。

04

看得见，随时叫停

工具栏角标提示「正在执行」，命令全程进审计日志，可申诉。一键暂停，AI 的命令立刻被拒，连接还在。

读需登录的页面

内网系统、要登录态才能看的页面——AI 用你的会话直接打开、读内容、抓数据。

🖱️

自动走流程

点按钮、填表单、多标签操作，一连串重复的浏览器活让 AI 替你跑完。

截图 · 导出 · 抓包

整页截图、导出 PDF、网络请求抓包——取证、归档、调试一步到位。

Python SDK，同步异步都有。标 需授权 的命令默认关闭，按用户、按域名开。

navigate打开 URL，新标签或复用当前
snapshot抓无障碍树，元素带 @e 引用
click按 ref 或选择器点击
fill填 input / textarea / 富文本
screenshot全页或元素级截图
evaluate需授权页面里跑任意 JS
network需授权抓包：start/stop/list/detail
upload上传文件，base64，≤5MB
save_as_pdf当前页导出 PDF
list_tabs列出本 session 的标签
find_tab按 URL 模式找标签
close_tab · close_session收尾，关标签或整组
你始终握着方向盘

安全是设计出来的，不是补上的

🔑

三级 Token

用户级 bb_usr_ 调命令、设备级 bb_dev_ 连扩展、管理级发配对码。全部哈希存储，互不越权。

🚪

高敏命令默认关

evaluate（跑 JS）和 network（抓包）默认禁用，需按用户、按域名白名单显式开通。

🧭

三层限流

按会话、按用户 24 小时、按设备并发分别设限，超了返回 429，挡住失控的调用。

📜

全程审计可申诉

每条命令落审计日志，你能用自己的会话查记录，对某条标「这条不对」发起申诉。

⏸

一键暂停

弹窗点暂停，AI 发来的命令立刻全被拒，但连接保持。想停就停，不用断开重配。

🚫

禁内部页

拒绝导航到 file: / chrome: / javascript: 等特权 URI，上传也限 5MB。
